# tinotax-pricing/src

Pricing implementation modules.

## Module Map

- `price_book.rs` loads and queries price observations.
- `missing.rs` exports missing price requirements.
- `manual_import.rs` imports accountant/manual prices.
- `provider.rs` fetches provider prices.
- `valuation.rs` applies prices to reviewed ledger events.
- `audit.rs` writes pricing audit outputs.
